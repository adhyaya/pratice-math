### Install dependencies

node requirement

version 12.\*.\*

```
npm install
```
Add .env.json for specific environment.

### Run Development Server

```
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


### Run Production Server

Create Build:

```
npm run build
```

Start Server:

```
npm start
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
